# Dotfiles

Dotfiles for macOS and Linux.

## Install

```bash
git clone https://github.com/johndobie/dotfiles.git ~/.dotfiles && ~/.dotfiles/install.sh
```

## What's included

- **Ghostty** - Terminal emulator
- **Nushell** - Modern shell
- **Starship** - Shell prompt
- **Micro** - Text editor
- **Node.js** + **pnpm** - JavaScript runtime and package manager
- **JetBrainsMono Nerd Font**

## Dry run

Preview what will be installed without making changes:

```bash
./install.sh --dry-run
```
